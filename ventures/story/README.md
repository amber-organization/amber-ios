# Story: Social media rebuilt for humans.

A verified-human social platform fighting AI slop, bots, polarization, and attention collapse, built on small circles, daily prompts, and reciprocity, with no ads, no dopamine loops, and no outrage algorithms. Starting with students and campus circles.

**Live demo:** [story-gamma-ten.vercel.app](https://story-gamma-ten.vercel.app)

> Screenshot

## Features

- **iOS-Style Interactive Web Demo:** floating iPhone mockup with a full navigable app shell, scale-adaptive to any viewport with a purple ambient glow aesthetic
- **Full App Screen Set:** Login, Onboarding, Home feed, Live Session, Coaching, Device Connection, Session Detail, Session Complete, and Profile screens
- **Verified Human Core:** one person, one face, one account. No bots, no AI-generated content; verification built into onboarding
- **Reciprocity-Based Feed:** daily prompt response plus Circle of real humans reply; the loop is structured around showing up for others, not infinite scroll
- **Anti-Attention Design:** no public like counts, no follower counts, no infinite scroll; the platform does not run on dopamine
- **Campus-First Distribution:** beachhead strategy targeting dense student networks and campus circles before general consumer rollout

## Non-Negotiables

1. Verified humans only. No bots
2. Anti-AI content stance. Highlights human capability
3. No ads, no selling attention, no selling data. Ever.
4. Does not run on dopamine. No infinite scroll or public counts
5. Campus-first. Students and campus circles as the first dense network
6. Reciprocity-based core loop. Daily prompt, Circle, real replies

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000), floating iPhone mockup with Story's launch screen.

```bash
npm run build   # production build
npm run lint    # ESLint check
```

**Deploy to Vercel:** Import at [vercel.com/new](https://vercel.com/new). Auto-detects Next.js, zero config.
