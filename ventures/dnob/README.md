# D-NOB

**Dedicated Network of Belonging**

*You shouldn't be alone when you're sick.*

An interactive iOS demo for D-NOB — a protected community platform designed to help children in hospitals form real friendships with other pediatric patients during treatment.

## The mission
Born from a founder's personal experience with childhood cancer, D-NOB exists because one friendship can change everything. Every child deserves the chance to find it.

## What D-NOB does
A safe, hospital-partnered connection system where young patients can chat, call, and build real friendships with peers going through similar experiences. Permissioned, moderated, and designed with families and hospital staff at the center.

## This demo
Built with Next.js 15 + TypeScript + Tailwind. iOS-style interactive mockup showing the core patient experience.
